Variable declaration

Variables in lx types are hard coded. This means that the type of the variable cannot be changed at runtime. This is because C++ is a statically typed language.
Also similiar to C++ you can define a variable without a value. This is useful for when you want to define a variable but you don't have a value for it yet. Note that the variable will be undefined until you assign a value to it.

int var1 = 0;
int var2; # Undefined

Types

Int and Float

Int and Float are both reprsentations of numbers. Int is a whole number and float is a decimal number.

int var1 = 0;
float var2 = 0.0;

# var3 will have a contents of 3 due to truncation
int var3 = 3.1459

# var4 will have a contents of 3.1459 because it is a float
float var4 = 3.1459

Ints are defined by the 'int' keyword and floats are defined by the 'float' keyword. You can also make it unsigned by adding the 'unsigned' keyword before the type. This means that the number cannot be negative.

Bool

Bool is a representation of a true or false value. It is defined by the 'bool' keyword.

The values are 'true' and 'false'. The first letter can be capitalized or lowercase. You can also use '1' and '0' to represent true and false.

# All variables store true
bool var1 = True;
bool var2 = true;
bool var3 = 1;

Char and String

Char is a single ASCII character. If you do not know what ASCII is find out here.
It is defined by the 'char' keyword.

String is a collection of characters. It is defined by the 'string' keyword. You can access the individual characters of a string by using the [] operator.

# Stores the letter 'a'
char var1 = 'a';

string var2 = "Hello";
char var3 = var2[0];

Void

Void is a type that represents nothing. It is defined by the 'void' keyword. It is used for functions that do not return a value.

You can also use void as a type for a variable. This means that the variable does not have a type. This is useful for when you want to store a pointer to any type (however it is recommended to use a base class pointer).
