LX C++ Preprocesor

This project is creating a preprocsessor for C++. This means lx code will be translated into c++ then compiled. This means it is easy to integrate into existing projects and can be used in any c++ project.

Syntax for the language can be found here

Custom Features of LX

Type Switches

Type Switches are a way to run a certain scope of code for the true type of a pointer. This is useful for when you have a base class pointer and you want to run a certain function for each derived class. More info can be found here: WIP

Easy Enum Conversion With Strings

Enums can be converted to strings and vice versa with ease. This is useful for when you want to convert an enum to a string for debug printing or storing in a file for a more readable format. More info can be found here: WIP

Extensive Macro Functionality

Macros in LX are more powerful than in C++. They automatically have information about the ammount of arguments they have. Other features can be found here: WIP

Function Decorators

Function decorators are a way to add functionality to a function without modifying the function itself. This is useful for when you want to add logging, timing, or other functionality to a function without modifying the function itself. More info can be found here: WIP
